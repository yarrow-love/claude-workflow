# Provenance — section stamps, session archive, /archive, /initialize

## Objective

Land the intercom-and-durability layer: per-section provenance stamps in the workflows, the transcript archive behind the stamps, and the two remaining commands. After this order, every handle stamped into a work doc is durable, citable, and consultable — the single-machine foundation the future cross-LAN intercom routes over.

## Steps

1. **Per-section provenance stamps** (adopting `26-07-05.per-section-provenance` as reconciled in order 1): add the self-stamp close-step to `work-order.md`, `work-order-automatically.md`, `execute-milestone.md`, and the planner/architect close-steps — every appended `<h2>` section carries `> <role>@<machine>/<session-id>` under its heading; a section revised by a different session appends a second stamp line. Settle the proposal's open forks per its leans: visible blockquote; transient specialists stamp the nearest resumable session with their own role in the role segment; `sessions:` frontmatter is *derived* from body stamps (dedupe), not hand-maintained in parallel.
2. **`bin/session-archive`** (bun) — archive one session: locate `~/.claude/projects/<cwd-slug>/<uuid>.jsonl`, gzip → `work/sessions/<machine>/<uuid>.jsonl.gz`, append a sha256 line to `work/sessions/MANIFEST`; idempotent re-archive (last write wins — transcripts only grow). Create `work/sessions/` with a README line (grammar pointer; `zgrep` recipe for citing `#message-id` fragments).
3. **`bin/consult`** (bun) — the restore-before-resume ladder: `bin/consult <role@machine/uuid> "<question>"` → transcript present locally → `claude -p --resume <uuid> --model fable` (fallback opus); absent → decompress from `work/sessions/` into the local project store → resume; archive miss → report the degrade path (documents + fresh dispatch). Machine mismatch on the handle → stop with a clear message (cross-LAN is deferred; the grammar already carries the routing key). Repoint the workflows' consult steps at the wrapper so they stay one-liners.
4. **`/archive` command** — the checkpoint sweep for "not active, not closed" stopping points: grep `work/` for `sessions:` entries and body stamps → verify each `@<this-machine>` handle exists in `work/sessions/` → backfill via `bin/session-archive` → archive the *current* session → report (archived / already-present / unreachable-other-machine). Doubles as the pre-migration step for moving to another machine (archive → sync repo → restore on the far side).
5. **`/initialize` command** — read-only environment audit, reporting a pass/fail table: `.claude/` matches `BASELINE.md` (files present; no unlisted strays without project justification); user-scope MCPs present (context7, playwright, chrome-devtools); hooks in `settings.json` resolve to real files; `work/PROJECT.md` exists and covers the supplement questions; root `CLAUDE.md` present; gitignore hygiene (secrets patterns, large-format files, `tmp/`); `work/` layout conforms to `work/README.md`. Audit only — every failure names the fix but performs nothing (`/configure`, deferred, is the repair counterpart).
6. **Optional SessionEnd hook** — `settings.json` hook invoking `bin/session-archive` on session end so `/archive` becomes verification rather than primary mechanism. Decide at execution (verify the hook payload carries the transcript path / session id); if deferred, note it in the follow-up `/configure` filing.
7. **Retention floor** — raise the CLI transcript `cleanupPeriodDays` in user settings; record as an operator recipe in `BASELINE.md`.

## Acceptance

- A dispatched specialist's appended section carries a valid stamp; `sessions:` frontmatter derivable from body stamps with no drift.
- `/archive` on this repo archives the series-architect session (`architect@play/a9ddd6b3-…`) and reports a complete store; `work/sessions/MANIFEST` verifies.
- End-to-end consult test: delete a transcript from the local store, run `bin/consult` against its handle → restore → resume → answer returns. (Use this series' own architect session as the test subject.)
- `/initialize` passes on this repo post-series — and, with a deliberately broken input (e.g. hook pointed at a missing file), fails with a report naming the fix.
- `zgrep` of a `#message-id` fragment against an archived transcript locates the cited turn (grammar round-trip proven).
