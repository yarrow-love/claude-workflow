# BASELINE's sanctioned-absence list is a strict subset of the gate's

## Bug Report

`.claude/BASELINE.md:49` ("**Paths that post-date the scaffold** — created by convention as work begins, so absence is *not* a gap and reference gates must sanction it") names **three** paths:

- `work/PROJECT.md`, `work/sessions/MANIFEST`, `src/_archive/`

`work/PROJECT.md:9` — this repo's own reference-resolution gate, and the shape exemplar a consuming repo copies — sanctions **seven**:

- the three above, plus `tmp/*`, `work/MISSION.md`, `work/todos/BACKLOG.md`, and `work/design/` contents (DESIGN.md, `system/`, `sessions/`)

**Symptom.** A consuming repo authors its quality gate at bootstrap step 3 using BASELINE as the specification. Taking BASELINE's list at face value, its gate goes red on four paths whose absence is legitimate and expected — `tmp/*` (runtime stream logs), `work/MISSION.md` (created at `/plan` mission tier), `work/todos/BACKLOG.md` (created at first `/triage`), and `work/design/` contents (created at the founding `/design`). Every one of these is a lazy-create the workflow itself performs later.

Observed live during M3 phase 3's reference-gate run: 4 paths reported unresolved across `.claude/**.md` — `src/_archive/`, `tmp/accomplish`, `tmp/transcript`, `work/todos/BACKLOG.md`. Three of those four are sanctioned by `work/PROJECT.md` but **not** by BASELINE.

**Where found.** `.claude/BASELINE.md:49` against `work/PROJECT.md:9`.

**Why it was not fixed in place.** `.claude/BASELINE.md` is final `main`-bound text closed by M3 phase 2; phase 3 is barred from doctrine authoring beyond its one sanctioned ADR append. Editing it would reopen a sibling phase's acceptance surface un-blessed. Confirmed by the milestone architect at phase 3's blessing.

**Mitigation already in place.** The close runbook carries the full 7-entry list inline, so the fold ceremony itself is unaffected.

**Disposition.** Carries forward — to a future mission, or to an operator-sanctioned post-fold hygiene commit on `main`. That routing is the close triage's call, not phase 3's.
